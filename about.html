Project For Create website

"Create a responsive and user-friendly website using HTML, CSS, and JavaScript that allows users to upload and edit images. The website should have the following features:
1. Image Upload: Users should be able to upload an image from their device.
2. Image Compression: Users should be able to compress the uploaded image to reduce its file size. Provide a slider or input field to adjust the compression level (quality percentage).
3. Image Rotation: Users should be able to rotate the image by 90 degrees clockwise or counterclockwise.
4. Image Cropping: Users should be able to crop the image by selecting a specific area.
5. Download Edited Image: After editing, users should be able to download the modified image.
Additional Requirements: • The website should have a clean and modern design using CSS.
• Use JavaScript for all interactive functionalities (e.g., compression, rotation, cropping).
• Ensure the website is mobile-friendly and responsive.
• Provide clear instructions or tooltips for each feature.
Note: Use modern JavaScript libraries like Cropper.js for cropping and Canvas API for compression and rotation."
